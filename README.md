# From Manual to Macro

Fiji/ImageJ macros for semi-automated collagen fibril diameter quantification from TEM images.

Associated publication:
Bhandari A, Mizuno N, Tufa SF, Sakai LY, Keene DR, Shalhub S.
From Manual to Macro: A Reproducible Fiji Workflow for Semi-Automated Collagen Fibril Diameter Quantification in Transmission Electron Microscopy.
Journal of Histochemistry & Cytochemistry. 2026.

Full publication available at:
https://doi.org/10.1369/00221554261440588

Usage
1.⁠ ⁠Install Fiji/ImageJ (recommended: v1.54p)
2. ⁠Download the macro: vEDS_Wand_VALIDATE_Diameter_MajorMinor_ELLIPSE_154p.ijm
3. ⁠⁠Open Fiji and install the macro: Plugins → Macros → Install…
4. ⁠⁠Confirm shortcut installation: Plugins → Shortcuts → List Shortcuts
5. ⁠⁠Press lowercase q to run the macro after ROI selection.
